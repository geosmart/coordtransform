CoordTransform 坐标转换
---
[wandergis/coordtransform](https://github.com/wandergis/coordtransform)的Java版。
提供百度坐标（BD09）、国测局坐标（火星坐标，GCJ02）、和WGS84坐标系之间的转换的工具模块。
